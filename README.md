TEDxGateway Event Planner
A fully functional event management web application built to simulate real-world operations of a large-scale TEDx event. Designed to demonstrate end-to-end event coordination workflows including speaker management, vendor tracking, guest registration, budget monitoring, and day-of scheduling.
Why I Built This
As part of my preparation for a role in event and project management, I researched how large-scale events like TEDxGateway are operationally managed. I built this tool to demonstrate my understanding of the end-to-end planning process — from vendor negotiation to guest experience to budget tracking.
Features
Dashboard

Live event stats — confirmed speakers, registered guests, budget used, days to event
Interactive pre-event checklist with completion tracking
Budget snapshot with visual progress bars
Quick notes section for reminders and ideas

Speaker Management

Speaker cards with name, talk topic, and confirmation status
Add new speakers dynamically via form
Visual status badges — Confirmed, Pending, Cancelled

Vendor Tracker

Full vendor database with category, contact, amount, and contract status
Add new vendors with one click
Categories include AV & Tech, Catering, Branding, Decor, Transport, Security

Guest List

Guest registration table with organisation, email, and guest type
Live search to filter guests instantly
Guest types — General, VIP, Sponsor

Budget Manager

Full expense breakdown by category
Budget health tracker showing percentage used
Income tracking — sponsor revenue and ticket sales
Net position calculator

Event Timeline

Pre-event milestones with completion indicators
Full day-of schedule from setup to closing
